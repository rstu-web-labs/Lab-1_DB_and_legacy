SQL-скрипт выполняет следующие действия:

1. Создание базы данных: `CREATE DATABASE life_on_wheels;` создает новую базу данных под названием `life_on_wheels`.

2. Создание схемы для сырых данных: `CREATE SCHEMA raw_data;` создает новую схему под названием `raw_data`.

3. Создание таблицы для сырых данных: Этот блок кода создает таблицу `sales` в схеме `raw_data`. 
Таблица содержит следующие столбцы: `id`, `auto`, `gasoline_consumption`, `price`, `date`, `person_name`, `phone`, `discount`, `brand_origin`.

4. Загрузка данных из CSV-файла в таблицу sales: 
`COPY raw_data.sales(auto, gasoline_consumption, price, date, person_name, phone, discount, brand_origin) 
FROM '/cars.csv' 
DELIMITER ',' 
CSV HEADER;` 
загружает данные из файла CSV в таблицу `sales`.

5. Создание схемы: `CREATE SCHEMA car_shop;` создает новую схему под названием `car_shop`.

6. Создание таблицы в схеме: Этот блок кода создает таблицу `brands` в схеме `car_shop`. 
Таблица содержит следующие столбцы: `brand_id`, `brand_name`, `brand_origin`.

7. Создание таблицы cars: Этот блок кода создает таблицу `cars` в схеме `car_shop`. 
Таблица содержит следующие столбцы: `car_id`, `brand_id`, `model`, `color`, `gasoline_consumption`.

8. Создание таблицы sales: Этот блок кода создает таблицу `sales` в схеме `car_shop`. 
Таблица содержит следующие столбцы: `sale_id`, `car_id`, `customer_id`, `sale_date`, `price`, `discount`.

9. Создание таблицы customers: Этот блок кода создает таблицу `customers` в схеме `car_shop`. 
Таблица содержит следующие столбцы: `customer_id`, `full_name`, `phone`.

10. Заполнение нормализованных таблиц данными: Этот блок кода заполняет таблицы `brands`, `cars`, `customers` и `sales` данными из таблицы `sales` в схеме `raw_data`.

11. Запросы для аналитики: Этот блок кода содержит несколько SQL-запросов, которые выполняют различные аналитические задачи, 
такие как вычисление процентного соотношения автомобилей без указания расхода топлива, вычисление средней цены автомобилей по брендам и годам, 
вывод списка купленных автомобилей для каждого пользователя, вычисление максимальной и минимальной цены автомобилей по странам происхождения бренда и подсчет количества пользователей из США.
